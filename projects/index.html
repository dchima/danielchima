We set out to design an interactive gaming environment
          to demonstrate the versatility of the DE1-SoC device.
          the embbeded system utilizes various hardware periph-
          erals on the board.
          The entire system was designed using the C programming
          framework.

          The environment consists of two games; 'Hero'.
          which uses the avengers marvel characters, and a classic
          'Snake' game.

          Hero is a one player shooter designed to test a player's
          speed and reaction time. The player had five lives and
          is expected to dodge incoming enemmy ships. A life is
          lost if an enemy ship collides with the player avatar.

          Snake is a single player classic (remember nokia snake)
          in chich the player manoeuvers a constantly moving Snake
          within its environment. The snake grows as it eats food
          randomly spawned, and crashed when it collides with an
          obstacle or itself.



        
          To actualize our goals of creating a gaming environment on a
          bare metal system, several code architectures were utilized in
          the process. For timing operations, we made use of HPS private
          timers for task scheduling. HPS_IRQ interrupts were employed
          for control of the audio functions and the I/O peripherals
          of the system.

          An object oriented programming approach
          was also used to create, and populate the environment with, enemmy
          ships and food objects (using C struct).
