We set out to design an interactive gaming environment
          to demonstrate the versatility of the DE1-SoC device.
          the embbeded system utilizes various hardware periph-
          erals on the board.

          The gaming environment was programmed using C, and it
          consists of two games; 'Hero'. which uses the avengers
          marvel characters, and a classic 'Snake' game.
